Sign in to reach your ledger data

This admin login protects the live PostgreSQL-backed expense and income APIs. Once authenticated, the dashboard connects to the database and unlocks the full app.

How it works

Protected DB access

The API now expects an authenticated admin session before it will read or write expenses and income.

  • CredentialsThe login checks the ADMIN_USERNAME and ADMIN_PASSWORD values from your environment.
  • SessionA signed HTTP-only cookie keeps the admin session off the front end.
  • SecretThe SESSION_SECRET value signs and verifies that cookie on every protected request.
